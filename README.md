# image-forge-ui-craft

> AI image generator UI prototype.

![Status: Experiment](https://img.shields.io/badge/Status-Experiment-yellow)

## What It Does

UI prototype for an AI image generation tool. Enter text prompts and see a placeholder for generated image results. Frontend-only experiment with no backend implementation.

## Tech Stack

`React` `Vite` `TypeScript` `shadcn/ui` `Tailwind CSS` `TanStack Query`
